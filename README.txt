По ссылке ниже можно зайти на сайт, который является решением тестового задания.
https://saftaphp.000webhostapp.com/main.html
Открыв ссылку вы можете увидеть схему ячеек, необходимо прокликать на ячейки, которые вы бы хотели объеденить в область, затем выбрать горизонтальное выравнивание, вертикальное выравнивание, цвет текста, цыет фона. Затем в поле ввести текст. После этого нажать ок. Если необходима еще одна область повторяем действия. После завершения жмем "сохранить"  получаем сгенерированный результат. По поводу реализации: логика выполнена на PHP, отрисовка с помощью HTML,CSS, также использовал jQuery.
